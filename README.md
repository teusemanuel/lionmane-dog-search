# lionmane-dog-search
Application to search for Breeds

## Run Application on development
open folder `web-app`
- 1º execute 
```
npm install
```
- 2º execute 
```
npm start
```
- 3º open a broser on url [http://localhost:4200](http://localhost:4200)

## Build Application from production
open folder `web-app`
- 1º execute 
```
npm install
```
- 2º execute 
```
npm run build-prod
```
- 3º open a broser on url [http://localhost:4200](http://localhost:4200)
